📄 README - Proyecto Web Mundial 2026
🌍 Descripción del Proyecto
Este proyecto consiste en el desarrollo de una página web informativa inspirada en el formato de Wikipedia, enfocada en la Copa Mundial de Fútbol 2026.
El objetivo principal es mostrar información detallada de los países participantes, incluyendo:
Datos generales del país
Historia futbolística
Plantel actual
Formación táctica
La página está diseñada utilizando HTML y CSS, priorizando un diseño claro, ordenado y fácil de navegar.

🎯 Objetivos
Crear una interfaz visual atractiva y organizada
Practicar maquetación web con HTML
Aplicar estilos y diseño con CSS
Implementar componentes reutilizables como tarjetas y carruseles

🧩 Funcionalidades
🏠 Página principal con introducción al Mundial 2026
🌎 Sección de países participantes
🃏 Tarjetas individuales por país
📋 Información detallada de cada selección
🎞️ Carrusel de imágenes (solo HTML y CSS)
📌 Menú de navegación fijo

🛠️ Tecnologías utilizadas
HTML5
CSS3
(No se utilizó JavaScript en algunas funcionalidades como el carrusel, que fue implementado solo con CSS.)

🎨 Diseño
El diseño de la página se basa en:
Uso de colores relacionados al fútbol (verde, blanco, negro)
Tarjetas visuales para organizar contenido
Navegación simple y accesible
Distribución centrada y responsive (adaptable)

🚀 Cómo usar el proyecto
Descargar o clonar el repositorio
Abrir el archivo index.html en el navegador
Navegar por las distintas secciones del sitio

📌 Posibles mejoras
Agregar JavaScript para mayor interactividad
Implementar base de datos con información dinámica
Mejorar el diseño responsive para dispositivos móviles
Agregar buscador de países
